WebChat — простий груповий чат

Мета роботи — створити простий груповий чат з використанням React на фронтенді та WebSocket-сервера на Node.js.

Функціонал застосунку

Підключення декількох користувачів одночасно
Реальний час надсилання та отримання повідомлень
Введення свого імені перед входом у чат
Проста та зрозуміла структура коду

Технології
Frontend:
React
JavaScript
WebSocket API

Backend:
Node.js
Бібліотека ws

Як запустити проєкт
Запуск сервера
cd chat-server
node server.js


Сервер запускається на:

ws://localhost:8080

Запуск клієнта
cd chat-client
npm start

Клієнт відкриється в браузері за адресою:
http://localhost:3000
